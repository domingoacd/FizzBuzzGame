0

  • There is going to appear a random number, between 0 and 100, on screen.
  • You have to write the next number, but following some rules:
    • If that number is divisible by 3, you have to write fizz
    • If the number is divisible by 5, then you write buzz
    • If the number is divisible by 5 AND by 3, then you have to write fizzbuzz
    • For the rest, you just type the number
  • The game ends when you run out of time to write your answer (5 seconds)
